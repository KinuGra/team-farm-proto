## 概要

このプロジェクトはUnityで作成されたプロジェクトです。

---

## 使用Unityバージョン

- Unity **6000.3.10f1**

---

## セットアップ方法

### 1. リポジトリをクローン

```bash
git clone <repository-url>
```

### 2. Unity Hubで開く

1. Unity Hubを起動
2. 「Add」または「Open」をクリック
3. このプロジェクトフォルダを選択

---

## フォルダ構成

- `Assets/` : ゲームの素材・スクリプト
- `ProjectSettings/` : プロジェクト設定
- `Packages/` : 使用パッケージ

---

## 注意事項

- `Library/` フォルダはGitに含めない（自動生成されるため）
- 初回起動時は読み込みに時間がかかります

---

## トラブルシューティング

### プロジェクトが開けない

- Unityのバージョンを **6000.3.10f1** に合わせる

### エラーが出る

- `Library` フォルダを削除して再起動
